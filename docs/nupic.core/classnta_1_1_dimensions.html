Detailed Description

A Dimensions object represents the dimensions of a Region It is vector of ints with a few methods for convenience and for wrapping.

A node within a region is identified by a Coordinate, and the Coordinate<->index mapping is in x-major order, i.e. for Region with dimensions [2,3]: 

[0,0] -> index 0
[1,0] -> index 1
[0,1] -> index 2
[1,1] -> index 3
[0,2] -> index 4
[1,2] -> index 5
