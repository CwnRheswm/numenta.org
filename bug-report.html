How to Report Bugs

If you are having problems installing NuPIC, the best place to get help is on the nupic-discuss mailing list or the #nupic IRC channel.

To file a bug against the python nupic project, you may create a new ticket here. Bugs for the nupic.core C++ project should be filed here. Please try to be as descriptive as possible and include error messages, steps taken before the error was encountered, and any additional details you think might help.
